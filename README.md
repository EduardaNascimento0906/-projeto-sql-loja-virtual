# Projeto SQL – Loja Virtual

Este projeto simula o banco de dados de uma loja virtual, com clientes,
produtos, pedidos e relatórios de negócio.

## Tecnologias
- MySQL 8.0
- SQL

## Estrutura do Banco
- Clientes
- Produtos
- Pedidos
- Itens do Pedido
- Localização (UF e Cidade)

## Exemplos de Consultas
- Total gasto por cliente
- Produtos mais vendidos
- Faturamento da loja

Projeto desenvolvido para fins acadêmicos e portfólio.
